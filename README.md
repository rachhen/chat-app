# Chat App
មុននឹងអាចដំណើរការកូដនេះបានអ្នកទាំងអស់គ្នាត្រូវតែដំឡើង [Nodejs](https://nodejs.org/en/)  ជាមុនសិន។ មើល [demo](https://rachhen-chat-app.herokuapp.com)

    git clone https://github.com/rachhen/chat-app.git
    cd chat-app
    npm i
    npm start
    or
    yarn 
    yarn start
